Methodology
06 — Methodology & Sources

How This Report Was Built

Data Sources

DataForSEO Labs — Keyword suggestions, keyword ideas, search volume (national + Sarasota DMA geo-targeted), keyword difficulty, CPC, competition index, search intent classification.

DataForSEO SERP API — Live Google SERP scrape for 3 query variants, geo-targeted to Sarasota FL. People Also Ask expansion to 3-click depth (yielding 13+ unique questions).

Google Ads API — Local search volume cross-referenced at Sarasota DMA level (location code 1015192) for accurate local demand sizing.

Queries Analyzed

"custom home builder Sarasota FL"
"best home builder Sarasota"
"cost to build a custom home Sarasota Florida"

Volume Methodology

National volumes from DataForSEO Labs (US-wide). DMA volumes from Google Ads API geo-targeted to Sarasota, FL metro. Where both available, DMA volume is the planning number — national volume shown for context only.

Intent Classification

Search intent from DataForSEO's ML classifier: commercial (comparison shopping), transactional (ready to act), informational (learning), navigational (seeking specific site).

Limitations & Caveats

Search volumes are 12-month averages and may not reflect real-time demand shifts. CPC values represent current Google Ads auction prices and fluctuate. Keyword difficulty scores are relative (0–100 scale) and should be validated against actual SERP authority profiles. Local Pack rankings are volatile and depend heavily on proximity, GBP completeness, and review velocity.
